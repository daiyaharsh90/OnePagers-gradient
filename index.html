The Theme

Make it your own! Be sure to read the README.md file to get started.

74 Gradients

Choose from 74 different gradients. Want to flip it? Add -inverse to the end of its class name.

For example, change purple-bliss to purple-bliss-inverse.

Icons Included

Choose from 675 icons in Font Awesome 4.7.0

View the complete list of icons here.

100% Responsive

Built using Bootstrap 3, this template scales to look beatiful on any size display.

Check the README.md document to see how to make your images responsive.

All the Icons!

OnePagers Gradient theme comes with Fontawesome 4.7.0 by Dave Gandy.

Use any icons from the complete set of 675 icons - go wild!

It's as simple as including <i class="fa fa-cogs" aria-hidden="true"></i>.
